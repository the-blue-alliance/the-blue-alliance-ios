Hey y'all - welcome to v3.4.0 (shoutout to my GRR friends + M&J). This update contains a lot of missing parity features. The Blue Alliance is built and maintained by volunteers - help us build The Blue Alliance for iOS at https://github.com/the-blue-alliance

Features:
* Added support for ALL playoff types in the Event Matches list - including double elimination brackets
* Playoff/elimination match cells now show Alliance Number (A1, A2, etc.) in the match cell
* Subdivision names now show in the match cells for matches on finals fields (ex: Einstein)
* Match cells now show RP dots for 2023, 2024, 2025, and 2026 season matches.
* Event Insights for 2023, 2024, and 2025 are now supported in the app.
* Match Breakdowns for 2017–2025 now split auto / teleop / overall instead of just showing the overall total.
* Ranking Point rows in breakdowns will now only show for quals matches where they apply.
* myTBA now only shows fully loaded cells - no more placeholder cells. Matches will no longer show in myTBA views.
* Event Stats (OPR, DPR, CCWM) now has a Team # sort option
* Long event names now wrap to multiple lines in Search, MyTBA, and event lists instead of being truncated
* New loading view for the Team header view - should be less jumpy and more pretty
* B teams are now properly supported for offseason events, will link to their parent team on tap

Bug fixes:
* Restored team nicknames on District Rankings, Awards, District Points, and Team@Event Stats (were falling back to "Team N")
* Restored event names on Team@District → Summary and Breakdown (rows / section headers were showing raw keys like "2026mibat")
* 4-column insights cells no longer appear tappable
* Fixed 2026 insights rendering when average win score is missing
* Search now allows for using the year in the search string
* Team page no longer shows a doubled "Team N / Team N" header when a team has no nickname
